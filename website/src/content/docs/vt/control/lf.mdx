# Linefeed (LF)

<VTSequence sequence="LF" />

This is an alias for [index (IND)](/docs/vt/esc/ind).

If linefeed mode (mode 20) is enabled, perform a
[carriage return](/docs/vt/control/cr) after the IND operation.
