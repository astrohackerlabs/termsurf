# Terminal API (VT)

> The Terminal API documentation is adapted from [Ghostty](https://ghostty.org)'s VT docs, used under the MIT license (see the repo `NOTICE`). TermSurf's terminal (Ghostboard) is a Ghostty fork and inherits its VT engine; product-specific details that still name Ghostty are being verified against TermSurf.

Programs running in terminals use **control sequences** to interact with the
terminal. Control sequences are a series of special characters that the terminal
interprets as commands. These are sometimes referred to as **VT sequences** or
**escape codes**.

For a full list of supported sequences, see the
[reference](/docs/vt/reference). For a conceptual overview
of control sequences, see the [concepts page](/docs/vt/concepts/sequences).
The current page will give an overview of Ghostty's control
sequence support.
