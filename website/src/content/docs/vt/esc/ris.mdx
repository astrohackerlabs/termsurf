# Full Reset (RIS)

<VTSequence sequence={["ESC", "c"]} />

The full reset operation does the following:

- Set the cursor shape to the default
- Reset the scroll region to the full screen
- Disable left and right margin mode (mode 69)
- Disable origin mode (mode 6)
- Unset cursor foreground and background colors
- Reset charsets to the default
- Reset cursor key mode (DECCKM)
- Reset disable keyboard input (KAM)
- Reset [application keypad mode](/docs/vt/esc/deckpnm)
- Reset xterm keyboard modifier state to the default
- Disable cursor protected attribute
- Disable any protected area
- Reset all mouse tracking modes
- Reset tabstops to default
- Enable send-receive mode (mode 12)
- Reset backspace sends delete (mode 67)
- Return to the primary screen and clear it
- Move the cursor to the top-left corner
- Reset the pending wrap state
- Reset saved cursor state
